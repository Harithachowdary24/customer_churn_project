# Customer Churn Prediction

Built an end-to-end machine learning model to predict customer churn using Python and Scikit-learn.

## Tools
Python, Pandas, NumPy, Scikit-learn

## Workflow
Load Data  
Clean Data  
Encode Categorical Columns  
Train/Test Split  
Scale Features  
Train Logistic Regression  
Evaluate Model  
Save Model  

## Results
Accuracy: 81%  
ROC-AUC: 0.84
